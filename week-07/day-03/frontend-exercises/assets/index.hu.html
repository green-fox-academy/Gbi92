REST Backend API - Frontend

Ez a frontend alkalmazás a végpontjaid kipróbálásához készült. A kipróbáláshoz nem szükséges, hogy módosítsd a frontend.js vagy az index.html fájlokat.

Ez a példa jól demonstrálja, hogyan zajlik a kommunikáció egy alkalmazás frontend-je és backend-je között. Mivel a frontend teljesen kész van és működőképes, így neked a feladatod már csak az, hogy a backend-et megírd és együttműködésre bírd őket.

A frontend.js fájl betöltése sikertelen. Kérlek ellenőrizd:
  • Jó static mappát használsz?
  • Létezik frontend.js nevű fájl?
